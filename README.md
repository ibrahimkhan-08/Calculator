# Modern Calculator

A modern calculator application built with Python and Tkinter. This project features a dark-themed user interface, calculation history tracking, keyboard support, and automatic history saving with date and time stamps.

## Features

- Modern dark-themed interface
- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Calculation history
- Date and time tracking for every calculation
- History saved automatically to a text file
- Clear history option
- Backspace functionality
- Keyboard support
- Lightweight and easy to run
